== Description ==

Keep your WordPress safe? There's an app for that, finally!

With hellosys you'll be able to control your WordPress's installation and performing the most important security tasks like:

- updating core
- updating plugins
- check if your website is up
- put your website in maintenance mode
- block users and reset their passwords

You'll also be able to track your websites's most important vital signs like:

- time to first byte (TTFB)
- time to last byte (TTLB)
- homepage's page size
- folder size (the space in your hosting your website is taking)

This plugin allows you to connect your website with the app through our intermediate and secure API server, to provide all of the above features quickly and securely.
